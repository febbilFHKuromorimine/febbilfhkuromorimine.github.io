Bab 42: Effort Lebih untuk Mengupdate Wangsaff

Update wangsaff memang sangat sya perlukan agar aplikasinya tetap berjalan, dan sya sering overthinking jika versinya sudah terlalu jadul, dikhawatirkan logout akunnya.

Sebelum mengupdate, ada baiknya untuk mengosongkan penyimpanan terlebih dahulu.

Ada 2 metode yg bisa sya lakukan:

Cara pertama: download di hp nya langsung di Playstore, namun wajib hapus data Playstore nya dulu (Playstore jadul tombol download masih mengotak warna hijau), dikarenakan suka "tertunda", jika di hapus data dulu baru mau, eh masih aja "kebegal" kek disuruh hapus app, padahal udah lega penyimpanannnya.

Cara kedua: kirim hasil update dari hp yg udah update via Playstore. Dan tetap aja masih sama, itupun ngirimnya dari Zenfone Max M1 (Sopod).

Jika kedua cara normal masih belum bisa, alternatif lain ya install apk (download apk nya di apkpure atau yg lainnya, dan sesuaikan arsitektur hp nya, yg masih 32 bit pilihnya armeabi-v7a dan yg udah 64 bit pilihnya arm64-v8a).

Sya coba restart lagi masih sama, dan akhirnya Shareme bisa terpasang, selanjutnya kirim apk update nya dari Zenfone Max M1, dan masih sama pemasang paket nya blank hitam dulu, dan akhirnya bisa terinstall.
